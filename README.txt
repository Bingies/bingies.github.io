Product Description:

Binge Hub is reinventing how everyone watches their favourite T.V shows.
With BingeHub, users can stream their favourite T.V shows in real time,
with other users at the same time. This allows users to communicate between 
eachother while they are watching the show, so that they can discuss plot 
lines, character developments and themes; in real time! There is also a forum 
section of BingeHub so that users can discuss certain points of any show with 
others of similar interest. You will be able to create a new thread on what you
want to discuss or comment on an existing thread that interests you!
